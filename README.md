# p2p-marketplace-safe-auth

This repository is focused on Safe (formerly Gnosis Safe) multi-sig integration for a Peer-to-Peer (P2P) marketplace. 

## Overview
The goal of this project is to provide a secure and decentralized authentication and transaction authorization layer for P2P marketplaces using Safe.

## Key Features (Planned)
- Safe Authentication: Users can log in and authenticate using their Safe multi-sig wallets.
- - Transaction Authorization: Multi-sig approval flow for marketplace transactions.
  - - P2P Marketplace Integration: Scaffolding for a P2P marketplace that leverages Safe's security features.
   
    - ## Project Structure
    - - contracts/: Core smart contracts for the marketplace and Safe integration.
